= 챌린지 API 문서

== 열거형 타입 항목 설명

이 문서에서는 챌린지 관련 API에서 사용되는 다양한 열거형 타입의 항목들을 설명합니다.
아래의 각 열거형 타입은 API 요청 및 응답에서 사용될 수 있으며, 각 항목은 Name과 Description으로 구성되어 있습니다.

- Name: 코드에서 사용되는 열거형 상수의 이름입니다.
- Description: 열거형 상수에 대한 설명입니다..

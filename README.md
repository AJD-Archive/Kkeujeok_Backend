## 🔔 소개

끄적끄적은 일정을 시작 전, 진행 중, 완료의 3가지 상태로 관리할 수 있는 일정 관리 플랫폼입니다. 개인 일정뿐만 아니라 팀 일정을 효과적으로 관리할 수 있으며, 다른 사람들이 여는 챌린지에 참여하여 협업과 동기부여를 지원합니다. 사용자는 드래그 앤 드롭으로 일정을 편하게 관리할 수 있습니다.
